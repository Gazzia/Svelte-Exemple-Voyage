# Svelte-Exemple-Voyage

SIMPLON - Exemple de projet Svelte pour la veille techno

## Pour initialiser:

```bash
	git clone https://github.com/Gazzia29/Svelte-Exemple-Voyage.git
	cd Svelte-Exemple-Voyage
```

**Avec npm :**

```bash
	npm install
	npm run dev
```

**Avec yarn :**

```bash
	yarn install
	yarn run dev
```
